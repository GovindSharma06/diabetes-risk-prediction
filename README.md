## 🚀 Overview

An end-to-end Machine Learning system for predicting diabetes risk using structured clinical and lifestyle data.

This project demonstrates a complete ML lifecycle:
- Data preprocessing & EDA
- Model comparison
- Hyperparameter tuning
- Threshold optimization
- Cross-validation
- Production deployment

---

## 🎯 Project Objective

To design and deploy a clinically interpretable, production-ready diabetes risk prediction system that:

- Maximizes recall while maintaining precision
- Uses threshold tuning for real-world decision support
- Demonstrates full-stack ML engineering capability
